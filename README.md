# eStore

### Introduce
일반 사용자는 상품을 장바구니에 담을 수 있으며, 관리자는 상품을 추가, 수정, 삭제할 수 있는 Spring framework 기반의 웹 애플리케이션입니다.

### Technical Skills
- Back-end: Spring5, Spring Security, Hibernate, REST API
- Front-end: AngularJS, Bootstrap
- Server: Tomcat9
- Database: MySQL8
